Print a graph or a certain part of it.

Note

The printer icon in this demo is enabled only if a default printer is configured.

Things to try

  • Create/change the graph as you like.
  • Resize and move the red dotted rectangle to change the area that is printed.
  • Disable Use Rectangle Content to ignore the red print rectangle and print the whole graph instead.
  • Toggle Scale down to fit content on one page to force scaling down the area to print on one page instead of multiple pages.
  • Toggle Scale up to fit content on one page to force scaling up the area to print to fill an entire page.
  • Enabling Center Content on Page will move the printed content to the center of the page if it completely fits on one. Otherwise, the content will be printed arranged on the top left corner.
  • Change the paper settings like the orientation, size or margins by clicking on Configure Paper....
  • Enabling Print Decorations will print a header and footer on each page. Note that those decorations need at least 50 pixel margins to the top and bottom.
  • Change the content margins. Notice the space that is reserved between content and page border when you change the values.
